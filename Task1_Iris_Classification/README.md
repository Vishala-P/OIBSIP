# Task 1: Iris Flower Classification

## Objective
To build a machine learning model that classifies iris flowers based on their morphological features.

## Dataset
The dataset is obtained from Scikit-learn’s built-in Iris dataset. It consists of three classes:
- Setosa
- Versicolor
- Virginica

## Features
The model uses the following features:
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## Algorithm
K-Nearest Neighbors (KNN) algorithm is used for classification.

## Implementation
The dataset is split into training and testing sets. The KNN model is trained using the training data and evaluated on the test data.

## Result
The model achieves high accuracy in predicting the correct class of iris flowers.

## Files
- Task1_Iris_Classification.ipynb

## Conclusion
The machine learning model successfully classifies iris flowers based on input features using the KNN algorithm.
